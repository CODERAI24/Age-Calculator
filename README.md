# Age-Calculator
Age calculator with live birthday countdown.

A responsive and interactive **Age Calculator web application** that calculates your age in real time and shows a **live countdown to your next birthday**, with **special secret messages for selected dates**.

 Features

* Calculate age in **Years, Months, and Days**
* Live ticking:

  * Weeks passed
  * Days passed
  * Hours passed
  * Minutes passed
  * Seconds passed
* **Live birthday countdown**
* **Password-protected secret messages** for special dates
* Fully **mobile-responsive**
* Clean, modern UI with soft blue & pink gradients

## Secret Dates Feature

Some specific dates display **special messages**.

* Certain dates are **password-protected**
* The countdown continues even while entering the password
* Password unlock happens instantly (no page refresh)

> Note:
> This is a frontend-only project. Passwords are for **personal/fun use**, not real security.

## How to Use

1. Open the website
2. Select your birth date
3. Click **Calculate Age**
4. View:

   * Your age
   * Live time passed
   * Countdown to next birthday
5. If the date is special, unlock the message
